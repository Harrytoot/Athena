# Investment Ontology v1.0

## Status

Ratified (2026-06-24)

## Purpose

本文定义 Athena 眼里投资世界的核心概念。

所有 Agent、策略、数据库、UI、Prompt 必须使用此处统一定义，不得自行解释。

---

## 1. Market States

| 概念 | 定义 | 量化标准 |
|------|------|---------|
| Bull Market（牛市） | 趋势向上，市场宽度好，资金持续流入 | 指数 > MA200, Advance/Decline > 1.5 |
| Bear Market（熊市） | 趋势向下，市场宽度差，资金持续流出 | 指数 < MA200, Advance/Decline < 0.7 |
| Range（震荡） | 无明确方向，在一定区间内波动 | 波动率低，突破失败率高 |
| Volatile（高波动） | 方向不定，但波动极大 | VIX 类指标 > 均值 2 倍标准差 |

## 2. Money Flow（资金流）

| 概念 | 定义 |
|------|------|
| **北向资金** | 通过沪/深港通进入 A 股的外资 |
| **主力资金** | 机构/大单资金流向 |
| **散户资金** | 个人投资者资金流向 |
| **板块轮动** | 资金从一个行业/概念流向另一个行业/概念 |

**资金流动原因模型：**

```
资金流动 = f（政策变化, 盈利预期变化, 估值对比, 风险偏好变化, 流动性环境）
```

## 3. Sector Rotation（行业轮动）

| 概念 | 定义 |
|------|------|
| **行业轮动** | 不同行业在不同经济阶段的相对强弱周期变化 |

**轮动驱动因素：**

1. 经济周期位置（复苏/过热/滞胀/衰退）
2. 政策支持力度
3. 盈利增速差异
4. 估值性价比
5. 资金偏好变化

## 4. Value vs Growth（价值 vs 成长）

| 概念 | 定义 |
|------|------|
| **价值（Value）** | 当前价格低于内在价值，安全边际大 |
| **成长（Growth）** | 未来盈利增长潜力大，当前估值可接受 |

**Athena 的判断标准：**

| 维度 | Value | Growth |
|------|-------|--------|
| PE | < 行业中位数 | 可高于中位数 |
| PB | < 2 | 可 > 3 |
| ROE | 稳定 | 上升中 |
| 盈利增速 | 稳定/持平 | > 20% |
| 股息率 | > 2% | 可不分红 |
| 行业阶段 | 成熟 | 成长/导入 |

## 5. Risk（风险）

| 概念 | 定义 |
|------|------|
| **系统性风险** | 影响整个市场的风险（政策、宏观、地缘） |
| **非系统性风险** | 影响个别公司/行业的风险（经营、财务） |
| **尾部风险** | 极端事件发生的风险（黑天鹅） |
| **流动性风险** | 无法以合理价格快速交易的风险 |
| **集中度风险** | 持仓过度集中的风险 |

## 6. Opportunity（机会）

| 概念 | 定义 |
|------|------|
| **机会** | 预期收益显著高于预期风险的情境 |

**机会识别条件（AND 关系）：**

```
机会 = 有利市场环境 
     + 行业/标的处于有利位置 
     + 多维证据支持 
     + 风险可控 
     + 有明确催化剂
```

## 7. Bubble（泡沫）

| 概念 | 定义 |
|------|------|
| **泡沫** | 资产价格严重偏离内在价值，主要由非理性情绪驱动 |

**泡沫识别信号：**

- 估值处于历史极值（> 95 分位）
- 情绪极度亢奋
- 新投资者大量涌入
- "这次不一样"叙事盛行
- 基本面不支撑当前价格

## 8. Trend（趋势）

| 概念 | 定义 |
|------|------|
| **趋势** | 价格在一段时间内持续朝一个方向运动 |

**趋势确认原则：**

1. 价格高于/低于关键均线
2. 高点/低点逐步抬高/降低
3. 成交量配合方向
4. 多时间框架共振

## 9. Quality（质量）

| 概念 | 定义 |
|------|------|
| **高质量企业** | 具有持续竞争优势、稳健财务、优秀管理的企业 |

**质量维度：**

| 维度 | 指标 |
|------|------|
| 盈利能力 | ROE, ROIC, 利润率 |
| 成长性 | 收入/利润增长率 |
| 财务健康 | 负债率, 现金流 |
| 竞争优势 | 护城河, 市占率 |
| 管理质量 | 管理层诚信, 资本配置能力 |

## 10. Margin of Safety（安全边际）

| 概念 | 定义 |
|------|------|
| **安全边际** | 买入价格与内在价值之间的缓冲空间 |

Athena 的安全边际计算：

```
安全边际 = (内在价值 - 市场价格) / 内在价值
要求: > 20%（价值股）或 > 10%（成长股）
```

## Concept Hierarchy

```
Investment Decision（投资决策）
├── Market Context（市场环境）
│   ├── Market State（Bull/Bear/Range/Volatile）
│   ├── Liquidity（流动性）
│   ├── Breadth（市场宽度）
│   └── Sentiment（市场情绪）
├── Asset（资产）
│   ├── Quality（质量）
│   ├── Valuation（估值）
│   ├── Growth（成长性）
│   └── Risk（风险）
├── Evidence（证据）
│   ├── Technical（技术面）
│   ├── Fundamental（基本面）
│   ├── Money Flow（资金面）
│   ├── Policy（政策面）
│   └── Sentiment（情绪面）
└── Action（行动）
    ├── Direction（买/卖/持有/观望）
    ├── Position Size（仓位大小）
    ├── Timing（时机）
    └── Confidence（置信度）
```
