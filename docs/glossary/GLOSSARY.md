# Glossary

## A

| 术语 | 定义 |
|------|------|
| ADR | Architecture Decision Record — 架构决策记录 |
| AES | Architecture Evolution Strategy — 架构演进策略 |
| Agent | 独立的 AI 分析单元，负责特定领域 |
| AMS | Athena Master Specification — 项目最高规格说明书 |
| Alpha | 超额收益 — 收益中不能被市场解释的部分 |

## B

| 术语 | 定义 |
|------|------|
| Backtest | 回测 — 用历史数据验证策略有效性 |
| Bear Market | 熊市 — 趋势向下的市场状态 |
| Beta | 市场敏感度 — 资产对市场波动的敏感程度 |
| Breadth | 市场宽度 — 上涨/下跌股票的数量比 |
| Bull Market | 牛市 — 趋势向上的市场状态 |

## C

| 术语 | 定义 |
|------|------|
| Confidence | 置信度 — AI 对自身输出的信心程度 |
| Constitution | 项目宪法 — Athena 最高法律文件 |

## D

| 术语 | 定义 |
|------|------|
| DDD | Domain-Driven Design — 领域驱动设计 |
| Decision Memory | 决策记忆 — 保存"为什么交易"的记录 |

## E

| 术语 | 定义 |
|------|------|
| Event Bus | 事件总线 — Agent 间通信的唯一渠道 |
| Evidence | 证据 — 支持/反对某个决策的数据点 |
| Evidence Graph | 证据图谱 — 多维证据的综合表示 |

## F

| 术语 | 定义 |
|------|------|
| Feature Store | 特征仓 — 统一存储和管理特征的组件 |

## I

| 术语 | 定义 |
|------|------|
| Investment Ontology | 投资本体 — Athena 对投资世界的概念定义 |

## K

| 术语 | 定义 |
|------|------|
| Knowledge Graph | 知识图谱 — 实体间关系的结构化表示 |

## M

| 术语 | 定义 |
|------|------|
| Meta Agent | 元调度器 — 动态调整各 Agent 权重的 Agent |
| Mock Provider | 模拟数据提供者 — 第一阶段的数据源抽象 |

## P

| 术语 | 定义 |
|------|------|
| PRD | Product Requirements Document — 产品需求文档 |
| Provider | 数据源抽象层 — 隔离外部依赖的接口 |
| Persona | 用户画像 |

## R

| 术语 | 定义 |
|------|------|
| Range | 震荡 — 无明确方向的市场状态 |
| RFC | Request for Comments — 重大设计提案 |

## S

| 术语 | 定义 |
|------|------|
| Sprint | 开发冲刺周期 |
| Strategy Marketplace | 策略市场 — 插件化策略注册系统 |

## V

| 术语 | 定义 |
|------|------|
| Vertical Slice | 垂直切片 — 每个 Epic 完成全栈闭环 |
| Volatile | 高波动 — 波动极大的市场状态 |
