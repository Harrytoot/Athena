# Macro Knowledge

## Purpose

存储关于宏观经济指标、货币政策、财政政策、经济周期等方面的已验证知识。

## Scope

- GDP、CPI、PPI、PMI 等宏观指标解读
- 央行货币政策传导机制
- 财政政策影响分析
- 经济周期识别与阶段特征
- 国际收支与汇率
- 地缘政治风险
- 全球宏观联动

## Allowed Document Types

- `KNOWLEDGE-*` — 单条已验证知识
