# Industry Knowledge

## Purpose

存储关于行业分析、产业趋势、竞争格局、商业模式等方面的已验证知识。

## Scope

- 行业分类体系（GICS、申万等）
- 行业生命周期与演进规律
- 产业链上下游关系
- 行业竞争格局分析框架
- 行业估值方法
- 行业轮动与风格切换
- 新兴产业识别
