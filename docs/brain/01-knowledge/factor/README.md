# Factor Knowledge

## Purpose

存储关于因子投资、风险因子、Alpha因子、因子构建与验证等方面的已验证知识。

## Scope

- 因子分类（价值、动量、质量、波动率、规模等）
- 因子构建方法论
- 因子有效性验证标准
- 因子衰减与拥挤度
- 因子组合与优化
- 风险模型
- Barra 风格因子体系

## Allowed Document Types

- `KNOWLEDGE-*` — 单条已验证知识
