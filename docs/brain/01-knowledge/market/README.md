# Market Knowledge

## Purpose

存储关于市场结构、微观机制、交易规则、市场参与者行为等方面的已验证知识。

## Scope

- 交易所规则与市场微观结构
- 市场流动性特征
- 定价机制
- 市场参与者的结构
- 交易时段与日历效应
- 跨市场联动机制
- 市场异常与套利限制

## Allowed Document Types

- `KNOWLEDGE-*` — 单条已验证知识
