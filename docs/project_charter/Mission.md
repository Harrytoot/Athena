# Mission

## What We Build

构建一个持续提升投资决策质量的人工智能系统。

不是"会买股票的 AI"，而是值得长期信任的 AI 投资伙伴。

## Why It Matters

- **个人投资者缺乏研究工具** — 机构有分析师团队，个人只有行情软件
- **信息过载而非信息不足** — 问题不是没数据，而是无法提炼洞察
- **决策无法追溯** — 没人记录"为什么买"，无法复盘，无法进步
- **AI 应该是伙伴** — 不是替代人，而是辅助人做出更好的决策

## How We Build It

### Documentation First
先写文档，再写代码。PRD → Architecture → Review → Implementation。

### Five Capabilities
每一版必须向五个能力之一交付价值：Observe / Understand / Reason / Recommend / Learn。

### Trustworthy AI
所有建议可解释。所有模型可追溯。所有变更可回滚。

### Continuous Learning
系统从每一次决策中学习。Hypothesis → Experiment → Backtest → Validate → Deploy。

## What We Don't Build

- 不会自动下单（必须人工确认）
- 不会替代投资判断（AI 辅助，人决策）
- 不会追求短期收益（长期决策质量优先）
- 不会做成封闭系统（插件化，可替换）

## Guiding Questions

每次开发前回答：

1. 它让投资决策更好了吗？
2. 它可解释吗？
3. 它能学习吗？
