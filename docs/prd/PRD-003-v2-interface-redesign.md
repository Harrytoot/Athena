## 一、产品定位与全局设计原则 (Global Principles)

**产品受限群体**：专业交易员、量化研究员。

**设计核心目标**：高信息密度（High Data Density）、决策导向（Decision-Centric）、操作沉浸感（Immersive Workspace）。

### 全局 UI 规范

#### 色彩系统 (Theme)
- 强制深色模式
- 背景色 `#080A10`
- 卡片背景 `#121621`
- 分割线 `#232838`
- 涨色：荧光青绿 `#00B8D9`
- 跌色：警示红 `#FF5630`

#### 排版约束 (Typography)
- 全局表单、图表轴线、数据展示区强制采用等宽字体 (JetBrains Mono)
- 行高压缩至 1.2 - 1.5
- 所有数值强制右对齐

#### 空间布局 (Layout)
- 彻底废弃侧边栏导航
- 全站采用"顶部极简导航 + Bento Grid (便当盒) 铺满全屏"的无滚动 (No-Scroll) 架构

## 二、核心页面模块详细设计

### 1. 顶部导航栏 (Top Navigation)

**功能定位**：全局路由切换与系统状态感知。

**布局与元素**：
- **左侧**：系统 Logo + Athena 缩写
- **中侧**：扁平化路由 Tab（大盘、自选、组合、策略、回测）。当前选中态底部高亮 `border-b-2 border-[#00B8D9]`
- **右侧**：核心系统状态指示灯
  - 环境切换: [实盘 Live] / [模拟 Paper]（目前锁定 Paper，绿色常亮）
  - 数据源延迟 (Latency): 显示当前行情接口延迟（如 12ms）
  - 时间戳: 交易所当前时间（带秒级跳动）

### 2. Dashboard & Market (市场全景总控)

**功能定位**：快速感知市场冷暖、板块轮动与核心宽基表现。

**布局与交互细节**：
- **顶部 (Ticker Tape)**：横向排列 3-4 个核心指数卡片。每张卡片高度极窄，包含：指数名、现价、涨跌幅、以及一张 20px 高的 Sparkline
- **左下方 (Market Breadth)**：市场宽度区。红绿渐变堆叠条形图，展示"涨停/上涨/平盘/下跌/跌停"家数分布。市场情绪综合评分 (0-100)
- **右下方 (Treemap 热力图)**：行业/概念板块透视。面积代表"成交额"，颜色深浅代表"涨跌幅"。支持 Hover 显示具体流入资金量

### 3. Watchlist (标的监控网格)

**功能定位**：自选股的高密度跟踪与策略信号初筛。

**布局与交互细节**：
- **极限 DataGrid**：类似 Excel/ag-Grid 的紧凑表格
- **必名字段**：代码、名称、现价、涨幅、量比、换手
- **策略信号列**：
  - RSI (14) 列：数值低于 30 标青色，高于 70 标红色
  - MA 状态列：微缩徽章（多头排列 Bull / 空头排列 Bear）

### 4. Portfolio (资产与风控透视图)

**功能定位**：监控持仓健康度与风险敞口。

**布局与交互细节**：
- **顶部指标条**：总资产、可用现金、当日 PnL、累计 PnL、实时最大回撤
- **中左侧 (Exposure Donut)**：环形图展示资产大类占比与行业集中度
- **中右侧 (Performance Chart)**：迷你组合净值曲线（叠加沪深 300 基准线）
- **下部 (Position Table)**：持仓明细表，含仓位占比 (%) 和浮盈/浮亏 (%)

### 5. Strategy Builder (策略研发工作台)

**功能定位**：可视化构建因子与交易逻辑。

**布局与交互细节 (三段式 IDE 布局)**：
- **左侧 (Node Library 抽屉)**：分类展示可用节点（数据源、技术指标、信号比较、风险控制、订单执行）。支持拖拽进入画布
- **中部 (React Flow 画布)**：带有暗夜网格线的画布，开启节点吸附。节点极度瘦身，深色磨砂材质
- **右侧 (Inspector 属性面板)**：默认隐藏。点击节点时从右侧滑出，提供表单编辑深度参数
- **操作流**：顶部工具栏提供 [保存拓扑布局] 和 [提交异步回测] 按钮

### 6. Backtest Report (回测报表 Tearsheet)

**功能定位**：深度归因与绩效展示。

**布局与交互细节**：
- **C位主图 (60% 屏幕空间)**：整合式时序图。上半部分策略净值 vs Benchmark 折线图；下半部分共用 X 轴的 Underwater Chart。支持十字光标联动
- **左侧 KPI 矩阵**：年化收益、夏普比率、最大回撤、胜率、盈亏比
- **底部面板**：月度收益热力图 (Monthly Return Heatmap)

## 三、交易执行闭环 (The Decision & Execution Loop)

### 1. Decision Center (决策中枢面板)

**出现位置**：个股分析页的右侧 30% 黄金视区。

**组件构成**：
- **结论聚合**：巨大的信号徽章（BUY, HOLD, SELL）与环形的综合置信度得分
- **因子共识 (Key Consensus)**：绿色 Check 图标打底的支持因子列表
- **关键风险 (Key Risks)**：红色 Warning 图标打底的阻碍因子列表
- **预期推演 (Expected Scenarios)**：Bull / Base / Bear 收益预期
- **行动区 (Action Bar)**：[ APPROVE ]、[ MODIFY ]、[ REJECT ] 三个按钮

### 2. Execution Ticket (交易执行舱)

**触发路径**：点击 Decision Center 的 [ APPROVE ] 按钮后，从屏幕最右侧 Slide-in 滑出浮层抽屉。

**组件构成**：
- **订单标的头**：股票名称代码 + 实时微型报价
- **算法单配置 (Algo Routing)**：MARKET / LIMIT / TWAP / VWAP 选择器
- **资金划拨 (Position Sizing)**：百分比滑块 (25%, 50%, 100%)
- **盘前成本分析 (Pre-Trade Analytics)**：黄色预警区域，展示滑点和市场冲击
- **最终提交**：[ SUBMIT TO PAPER TRADING ] 通栏按钮
